# IT Incident Management Dashboard

## Overview

This Power BI project analyzes IT service incidents and SLA performance using an ITSM dataset.

## Dashboard Pages

### Executive Overview
- Total Incidents
- Open Incidents
- Closed Incidents
- SLA Compliance %
- MTTR
- Reopened Incidents

### SLA Analysis
- SLA Met vs Breached
- SLA Compliance by Priority
- Reopened Incidents by Priority
- Resolution Time Analysis

### Incident Deep Dive
- Incident-level details
- Assignment Group Analysis
- Category Analysis
- Interactive filtering

## Tools Used

- Power BI
- DAX
- Data Modeling
- Power Query

## Key Insights

- 142K incidents analyzed
- 93.5% SLA compliance
- Moderate priority incidents dominate volume
- Reopen rate remains below 2%
